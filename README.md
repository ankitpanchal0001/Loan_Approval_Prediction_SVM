**Project Overview

This project builds a machine learning classification model to predict loan approval status based on applicant demographic and financial features.

**Objectives

Understand key factors influencing loan approval

Handle missing values and categorical variables

Train and evaluate an SVM classifier

**Tools & Technologies

Python

Pandas, Scikit-learn

Seaborn, Matplotlib

**Model Pipeline

Data cleaning & preprocessing

Feature encoding & scaling

SVM model with RBF kernel

Performance evaluation

**Evaluation Metrics

Accuracy score

Precision & Recall

Confusion Matrix

**Key Findings

Credit history is the strongest predictor

Income and loan amount influence approval decisions

SVM achieved balanced classification performance

**How to Run

Clone repository

Open notebook.ipynb

Run cells to train model
